# Cuestionario de descubrimiento — Familia Ponquesito

Propósito: recopilar la **información real** del negocio necesaria para
construir la base de conocimiento del asistente (Reto 1). Las respuestas de
este documento serán la única fuente para `src/knowledge/`; nada que no esté
aquí (o en material real adjunto) debe aparecer en las respuestas del
asistente.

**Cómo llenarlo:** responde debajo de cada `Respuesta:`. No hace falta
completarlo de una sola vez ni en orden; prioriza las preguntas `[MVP]`.
Si una pregunta no aplica, escribe "No aplica". Si no hay una respuesta
definida todavía, escribe "Sin definir" — eso también es información útil
(el asistente deberá decir que no lo sabe).

**Leyenda de prioridad:**

- `[MVP]` — Imprescindible para el MVP del Reto 1.
- `[ÚTIL]` — Útil, pero no bloqueante.
- `[FUTURO]` — Pendiente para una fase futura.
